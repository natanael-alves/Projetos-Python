## Simulador de Caixa Eletrônico em Python

Projeto desenvolvido com o objetivo de praticar lógica de programação e regras de negócio.

### Funcionalidades
- Consulta de saldo
- Depósito
- Saque
- Validações básicas

### Tecnologias utilizadas
- Python

### Objetivo do projeto
Projeto educacional para estudo de conceitos fundamentais de desenvolvimento backend.
